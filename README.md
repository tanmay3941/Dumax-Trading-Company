# Dumax Paints Website

## Files Included:
- index.html - Main HTML structure
- styles.css - All styling and animations
- script.js - Interactive JavaScript functionality

## How to Use:
1. Extract all files to the same folder
2. Open index.html in your web browser
3. All files must be in the same directory for proper functionality

## Features:
- Responsive design for all devices
- Interactive color visualizer
- Smooth animations and transitions
- Contact form with validation
- Mobile-friendly navigation

## Browser Support:
- Chrome (recommended)
- Firefox
- Safari
- Edge

For best experience, use the latest version of your browser.
